Code of Conduct for the bio-block project
=======================================

This repository follows the Contributor Covenant v2.1 to foster a welcoming,
respectful, and inclusive community. The full text is available at
https://www.contributor-covenant.org/version/2/1/.

Expected Behavior
-----------------
- Be respectful and considerate in all communication.
- Seek to be constructive and assume good faith.
- Follow maintainers' directions and respect project decisions.

Unacceptable Behavior
---------------------
- Harassment, discrimination, or exclusionary behavior based on identity.
- Threatening, abusive, or demeaning language or actions.
- Deliberate intimidation, stalking, or sustained disruption of community spaces.

Reporting Incidents
-------------------
Preferred: Open a GitHub Discussion in this repository and choose the
"Code of Conduct" category; include relevant details and context. If you
need confidentiality, state that in the report and maintainers will handle
it with discretion where possible.

Alternative: If you cannot use Discussions, contact the project maintainers
directly via email: `conduct@REPLACE-WITH-REAL-EMAIL.org`.

Maintainers: Replace `conduct@REPLACE-WITH-REAL-EMAIL.org` with a valid
contact address or reporting link before publishing this file.

Enforcement
-----------
Reports will be reviewed and investigated by the project maintainers. When
appropriate, consequences for unacceptable behavior may include warnings,
temporary or permanent bans from project spaces, and removal of
contributions.

Versioning
----------
This Code of Conduct is based on Contributor Covenant v2.1. It may be
updated by maintainers; updates will be documented in the project
repository.

Thank you for helping create a welcoming community.
